# meshinspector-backend

## Python Version Compatibility
This project supports Python 3.11 and 3.12. Python 3.12 is the recommended version, but 3.11 can be used as a fallback if you encounter library compatibility issues.

### Switching to Python 3.11
If you are using `uv`, you can specify the Python version for your environment:
```bash
uv venv --python 3.11
```
Or update the `.python-version` file to `3.11`.
